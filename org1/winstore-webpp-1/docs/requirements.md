# Application Requirements

## Functional Requirements  
- Users can view the list of stores participating in the display program.  
- Sales staff can upload actual display images.  
- Display data is automatically synchronized with the ERP system.  

## Non-functional Requirements  
- A simple and user-friendly interface on the web browser.  
- Data synchronization time must not exceed 5 minutes.  

## Security Requirements  
- Only authorized users can update data.  
- All update actions must be logged for system analysis.
